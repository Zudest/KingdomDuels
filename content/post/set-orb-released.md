**Orichalcos Barrier** is the new set released in Kingdom Duels on 2026-02-03.

#### Breakdown

- Codename: **[ORB]**
- It has **238 cards** from episodes [103 to 115](/story/wicked-arc/) of the Yugioh Scrolls project (it's the largest set so far!).
- It is focused on the adaptation of the "Waking the Dragons" arc with a few modifications here and there but overall retaining the same structure.
